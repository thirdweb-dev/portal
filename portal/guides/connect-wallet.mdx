const Connect = () => {
  return

## Choose the chain and wallet (provider and signer)

Now we need to setup our `Provider`. For this we will use the custom component of thirdweb, the `ThirdwebProvider`. Go ahead and import it.

```jsx:App.js
import { ThirdwebProvider } from "@3rdweb/react";
```

Next we need to select the chain and type of wallet we want to connect to.

With `supportedChainIds` we select the blockchain we want to connect with. We have to pass an `Id`. Every blockchain has an `Id`. We are using an `Id` instead of a `RPC url` here. (again click [here](/learn/connect-to-blockchain) if you donâ€™t know how a connection with blockchain works).

```jsx:App.js
const App = () => {
  // Polygon's chain Id is 137
  const supportedChainIds = [137];

```

With `connectors` we are selecting which wallets we want to support. If you want to learn more about our connect wallet component, click [here](/guides/add-connectwallet-to-your-website).

MetaMask `connector` setup ðŸ‘‡

We include the `ThirdwebProvider` in the return statement to wrap our app in the `ThirdwebProvider` , so that the provider information is accessible from anywhere.

`ThirdwebProvider` tags should be included at the highest level, usually in react that is app.js except in next. In next itâ€™s in \_app.js

So make sure you put your `Component` within the brackets of the `ThirdwebProvider`.

```jsx:App.js
	return (
    <ThirdwebProvider
      connectors={connectors}
      supportedChainIds={supportedChainIds}
    >
      <Connect /> /* our connect wallet button */
    </ThirdwebProvider>
  );
};
```
